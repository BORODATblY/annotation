Now when using a script in the terminal it is not necessary to download video from YouTube, just insert a link to the video (required in this format: https://www.youtube.com/watch?v=8JIqxLTBS5M (you need to double-click the link in the address bar) !!!!!!!). It also implements the ability to download video after slicing (in the terminal there will be a question: Download video? [Y / N] (answer y Y or n N), after pressing Enter, the video will automatically download to the folder with pictures. After the download is complete, all windows will close.
